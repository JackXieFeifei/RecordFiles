赢金币上限规则
1、玩家每局赢金币最高数量为自身持有金币。即在一局游戏中按照游戏规则计算赢取金币量大于自身持有金币量时，赢取的金币将限制为自身持有金币量，负方按比例承担输掉的金币。
2、当地主胜利时，若某农民的金币不足按规则计算出的金币量，将由另一农民承担不足金额。但后者付出金币总量不能超出按规则计算出的金币量。
3、农民胜利时，若地主金币不足按规则计算出的金币量，且农民之一的金币也不足规则计算出的金币量，则该农民少赢取的金币将分配给另一农民。后者获得的金币总量不得超出按规则计算出的金币量。
玩法介绍
1、一副牌54张，每人17张，底牌3张。牌不分花色。
2、该游戏由三人玩一副牌，分为一个地主和两个农民，共两个阵营对战，先出完的一方胜。
3、发牌：一副牌共五十四张，留三张底牌，其他发给三家。
4、叫分：发完牌后开始先叫分，每人只能叫一次，叫分可以选择"1分"、"2分"、 "3分" 或不叫，叫完后叫分最高者为地主。
5、出牌：首先将三张底牌交给地主，三张底牌所有人都能看到。由地主开牌，然后按逆时针顺序依次出牌。轮到你跟牌时，你可选择不要或按规则出牌。直至某一阵营牌出完就结束此局。
牌型介绍
特殊牌型
火箭：即双王（双鬼牌）
炸弹：四张大小相同的牌
普通牌型
单牌：单个牌
对牌：两张大小相同的牌
三张牌：三张大小相同的牌
三带一手：三张大小相同的牌＋一张单牌或一对牌
单顺：五张或更多的连续单牌（不包括 2 点和双王，不分花色）
连对：三对或更多的连续对牌（不包括 2 点和双王）
飞机：二个或更多的连续“三张牌”（不包括 2 点和双王）
飞机带翅膀：飞机 + 同数量的单牌或者对子，翅膀不能和任意的三张相同，翅膀和翅膀不得相同，大小王不能一起当翅膀
四带二：四张相同的牌 + 2张单牌或2个对子，不能同时带大小王，不能把另外一个炸弹拆开当2个对子带
牌型比较
火箭 ＞ 炸弹＞ 一般牌型（单牌、对牌、三张牌、三带一手、单顺、连对、飞机、飞机带翅膀、四带二）
说明：
一般牌型：牌型且张数相同才可以按牌点比较大小。 其中三带一、三带二、飞机带翅膀、四带二组合牌型，比较其相同张数最多的牌点数大小。
结算
底分：叫牌的分数。
倍数：初始为1，每出一个炸弹或火箭翻一倍。（火箭和炸弹留在手上没出的不算）
一局结束后：
地主胜：地主得分为2*底分*倍数。其余玩家各得：-底分*倍数
地主败：地主得分为-2*底分*倍数。其余玩家各得：底分*倍数
地主所有牌出完，其他两家一张都未出：分数 * 2。其他两家中有一家先出完牌，地主只出过一手牌：分数 * 2。
